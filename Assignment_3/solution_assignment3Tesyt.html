1. What Amsterdam will receive from tourist tax if the event lasts a week and you will have 30.000 visitors?

According to Hoeben (2023), the average tourist tax is 3,27 euros per night. The event lasts 7 nights, moreover we know that the total number of visitors is 30,000. 3,27 * 7 nights * 30,000 visitors = 686,700.00

Here a little code to calculate this:

In[]:
Total_nights = 7
Tourist_tax = 3.27
Total_visitors = 30000
Tax_received = Tourist_tax * Total_nights * Total_visitors

print(Tax_received)

686700.0

So the total amount of tourist tax Amsterdam will receive is 686,700.00.

3 Which street in Amsterdam has the most AirBnB apartments?

We used the python code that can be found below to find the street with the most AirBnB apartments. But running this code through the whole dataset of 8363 rows wasn't possible because it would run for hours. But we tested this code by adding iloc[:100]. The iloc[:100] isn't supposed to be in the code if you want to run it through the whole dataset. This means we only tried the first 100 rows and then the code worked. It means that it can be run for all the 8363 lines. The most common street name in the first 100 AirBnB listings is Oudezijds Voorburgwal.

In[]:
from geopy.geocoders import Nominatim
geolocator = Nominatim(user_agent="AMS")

#Create an empty 'streetname' column in the DataFrame
grid_Airbnb['streetname'] = ""

#Create a dictionary to store the counts of each streetname
streetname_counts = {}

#Iterate through the first 10 rows of your DataFrame
for index, row in grid_Airbnb.iloc[:100].iterrows():
    latitude = row['latitude']
    longitude = row['longitude']

    #Use latitude and longitude to get location information
    coordinates = f"{latitude}, {longitude}"
    location = geolocator.reverse(coordinates, zoom=17)

    #Extract the street name from the location information
    streetname = location.raw.get('address', {}).get('road', 'Unknown')
    grid_Airbnb.at[index, 'streetname'] = streetname

    #Count the occurrence of each street name
    streetname_counts[streetname] = streetname_counts.get(streetname, 0) + 1
#Find the most common street name
most_common_streetname = max(streetname_counts.get(streetname, key=streetname_counts.get))

print(f"The most common street name is: {most_common_streetname}")

The Dataset: Basisbestand Gebieden Amsterdam (BBGA) has information about the total number of Airbnb listings (houses or rooms) that were at least once offered on the airbnb platform once a year. That total is 28528 listings. But according to AirBnB the total listings is 8386. So that makes around 20.000 listings that are also used as normal housing.

Airbnb_listings = 8386

Not_airbnb_listed = BBGA_listings - Airbnb_listings
